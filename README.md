# Food Delivery API (Node + Express + PostgreSQL/Prisma)\n\n## Setup\n1. Create PostgreSQL DB `food_delivery`.\n2. Copy `.env.example` → `.env` and set `DATABASE_URL` + `JWT_SECRET`.\n3. Install deps: `npm i`.\n4. Generate Prisma client & migrate: `npx prisma generate && npm run prisma:migrate`.\n5. Seed (creates admin + sample data): `npm run prisma:seed`.\n6. Start dev: `npm run dev` → http://localhost:4000\n7. Swagger UI: http://localhost:4000/docs\n\n**Default admin**\n- email: `admin@example.com`\n- password: `Admin@123`\n\n## Endpoints\nSee `src/docs/openapi.yaml` or Swagger UI.\n\n## Tests\n`npm test` (Jest + Supertest)\n\n- Unit: register user, place order\n- Integration: register→login→order; admin status update\n
